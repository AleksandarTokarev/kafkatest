# How to start?
Start Docker Desktop, navigate to `docker` directory and run (up or down to start or bring down docker images)
```
docker-compose -f docker-compose.yml up
docker-compose -f docker-compose.yml down
```

Redis UI - Commander  
http://localhost:8081/  
Kafka UI  
http://localhost:8080/
